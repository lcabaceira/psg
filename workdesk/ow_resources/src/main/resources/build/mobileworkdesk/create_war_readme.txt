The .bat file creates a web archive, which can be deployed on an supported 
application server.

Requirement: The JAVA_HOME environment variable must point to the root
directory of a valid JDK installation, in order to find jar.exe.

On Linux/OSX, you can run the script with:
#sh create_mobileworkdesk_war.sh.sh
or
#bash create_mobileworkdesk_war.sh.sh
